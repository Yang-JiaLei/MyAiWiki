# Google AI Studio

## 是什么

Google AI Studio 是 Google 提供的开发者入口，用于测试 Gemini 模型、获取 API Key 并快速搭建原型。

## 关键事实

- 常用于申请或管理 Gemini API Key
- 适合做模型试玩、提示词验证和轻量开发
- 与 `gemini-cli` 和 Gemini API 场景高度相关
