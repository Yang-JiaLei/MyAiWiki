# 打卡说明

## 你应该怎么用

1. 每天新建一页，文件名用当天日期，例如 `2026-04-19.md`
2. 先填 frontmatter 里的 `date / mood / energy / focus / completed`
3. 再补三行正文：今天做了什么、遇到什么、今天收获什么
4. 晚上把 `completed` 改成 `true` 或 `false`

## 建议字段

```yaml
---
title: 2026-04-19 打卡
date: 2026-04-19
mood: 稳
energy: 7
focus: 主页改版与打卡系统
completed: true
win: 主页更像个人驾驶舱
---
```

## 你的最低使用标准

- 不要求长篇反思
- 只要求每天留下一个可回看锚点
- 如果很忙，只写 `focus + completed + win` 也可以

## 建议节奏

- 早上填 `focus`
- 晚上补 `completed` 和 `win`
