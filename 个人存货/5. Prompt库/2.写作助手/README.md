## 使用场景

1. **商业写作**：营销文案、产品说明、商业计划
2. **技术写作**：技术文档、API说明、教程编写
3. **创意写作**：故事创作、诗歌写作、剧本编写
4. **日常写作**：邮件撰写、报告编写、笔记整理

## 效果评估标准

- **语言质量**：语法正确性、表达流畅度
- **内容质量**：信息准确性、逻辑清晰度
- **创意程度**：创意新颖性、吸引力
- **风格适配**：符合目标风格和语气

## 最佳实践

1. **明确目标**：确定写作目的和受众
2. **提供范例**：提供参考风格和范例
3. **分步进行**：先大纲后细节，逐步完善
4. **多次迭代**：根据反馈进行多次修改优化
