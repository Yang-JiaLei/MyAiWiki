## 使用场景

1. **快速创建**：使用模板快速创建标准化的提示词
2. **质量保证**：确保提示词的结构和质量一致性
3. **团队协作**：统一团队使用的提示词格式和标准
4. **知识复用**：复用经过验证的有效提示词结构

## 模板特点

- **结构化**：清晰的逻辑结构和组成部分
- **可配置**：支持参数化和个性化配置
- **可复用**：可在不同场景中重复使用
- **可扩展**：支持根据需求进行扩展和修改

## 最佳实践

1. **选择模板**：根据需求选择合适的模板类型
2. **配置参数**：填写模板中的参数和变量
3. **测试验证**：测试模板的实际效果
4. **优化改进**：根据使用反馈优化模板
