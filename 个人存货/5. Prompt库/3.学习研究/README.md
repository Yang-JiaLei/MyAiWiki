## 使用场景

1. **学习规划**：制定学习计划、选择学习资源
2. **研究分析**：文献研究、数据分析、实验设计
3. **学术写作**：论文撰写、报告编写、学术交流
4. **知识管理**：知识整理、信息归类、笔记优化

## 效果评估标准

- **学习效果**：知识掌握程度和应用能力
- **研究质量**：研究设计的科学性和严谨性
- **写作质量**：学术表达的准确性和规范性
- **整理效率**：知识整理的效率和系统性

## 最佳实践

1. **明确目标**：确定学习或研究的具体目标
2. **分步进行**：将复杂任务分解为可管理的步骤
3. **持续反馈**：根据进展调整学习或研究策略
4. **系统整理**：建立系统的知识管理体系
