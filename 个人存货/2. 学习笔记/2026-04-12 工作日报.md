## 遇到的问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| bun 安装失败 | 缺少 unzip 依赖 | `apt install -y unzip` 后重装 |
| skills CLI 安装位置不明确 | .agents 目录未在文件系统找到 | 通过 `skills list -g --json` 确认已安装 |
| gemini-cli OAuth 无法在无服务器环境完成 | 需要浏览器打开和本地 HTTP 服务器 | 推荐使用 API Key 方式 |
| baoyu-fetch 依赖 Chrome | 环境未安装 Chrome | 文档中注明依赖，实际使用需安装 |
