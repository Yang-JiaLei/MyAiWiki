## 🔗 链接约定

所有内部链接格式：
```markdown
[[文件名|显示文本]]
```

示例：
- `[[Wiki百科/1. 实体/1.1 人物/andrej-karpathy.md|安德烈·卡帕西]]`
- `[[Wiki百科/2. 主题/2.2 技术/LLM.md|LLM 主题]]`
- `[[日志#最近操作]]`

## 🎯 使用流程

### 添加新源（如公众号文章）
1. 用户发送链接
2. LLM 读取内容（如果可访问）
3. 提取实体和主题
4. 更新/创建相关页面
5. 创建源摘要（`来源/微信/`）
6. 更新 `主页.md` 索引
7. 记录 `日志.md`

### 日常维护
- 阅读/编辑：直接在 Obsidian 中打开 vault
- 所有修改 FNS 自动同步
- LLM 通过 API 调用维护 Wiki
