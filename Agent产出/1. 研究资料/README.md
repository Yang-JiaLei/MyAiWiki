## 研究方法论

### 横纵分析法 (Horizontal-Vertical Analysis)

由数字生命卡兹克（Khazix）提出，融合了：
- 索绪尔的**历时-共时分析**（Diachronic-Synchronic）
- 社会科学的**纵向-横截面研究设计**
- 商学院**案例研究法**
- **竞争战略分析**

#### 三大核心轴

**1. 纵向分析（时间深度）**
- 追踪从诞生到当下的完整生命历程
- 以叙事故事呈现发展历程
- 关键事件、决策逻辑、演进路径
- 阶段划分和特征总结

**2. 横向分析（竞争广度）**
- 当下时间截面的竞品对比
- 系统性横向矩阵对比
- 识别差异化优势和定位
- 市场格局判断

**3. 交叉分析（交汇洞察）**
- 两条轴的交汇点
- 时机、策略、决策的独到洞察
- 判断和预测
- 行动建议

#### 研究流程

```
1. 前置准备
   ├── 明确研究对象
   ├── 确定研究维度
   └── 制定研究计划

2. 信息收集
   ├── 纵向信息（起源、发展、演进）
   ├── 横向信息（竞品、对比、矩阵）
   └── 补充信息（社区反馈、技术细节）

3. 分析整合
   ├── 纵向分析：完整时间线
   ├── 横向对比：竞品矩阵
   └── 交叉分析：独到洞察

4. 报告生成
   ├── Markdown 版本（版本控制友好）
   └── PDF 版本（排版精美，便于分享）
```

## 研究方法库

### 适用场景

横纵分析法适用于以下类型的研究：

- ✅ **产品深度分析**: 产品演进、定位、差异化
- ✅ **公司研究**: 发展历程、竞争格局、战略判断
- ✅ **技术调研**: 技术演进、竞品对比、选型建议
- ✅ **人物研究**: 人物背景、影响力、横向对比
- ✅ **概念分析**: 概念起源、发展、应用场景

### 输出物标准

研究报告应包含：
1. **执行摘要** (1页)
2. **纵向分析** (3-5页) - 时间深度
3. **横向对比** (2-3页) - 竞争广度
4. **交叉洞察** (2页) - 交汇点判断
5. **建议** (1-2页) - 行动指南
6. **附录** (1-2页) - 数据来源

**总篇幅**: 10-15页 (Markdown 30-50 KB)
