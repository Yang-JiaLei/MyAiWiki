## 注意事项

### ⚠️ 重要提醒

1. **hv-analysis 需要联网** - 深度研究需要联网搜索信息
2. **PDF生成依赖** - 需要安装 weasyprint 和 markdown
3. **触发词识别** - Skill 会根据用户输入自动触发
4. **人工审核** - AI 生成内容需要人工审核和修改

### 📝 使用建议

1. **深度研究**（hv-analysis）
   - 提供明确的研究对象
   - 确保研究对象有足够公开信息
   - 预留足够的搜索和分析时间

2. **公众号写作**（khazix-writer）
   - 提供清晰的写作主题和要点
   - 保持对内容的最终审核
   - 保留个人风格和观点
